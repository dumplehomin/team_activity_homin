####2.논점
**Mobile First**

>반응형 웹 구현 시 Mobile First 로 해야 할까?

####논의
1. 모바일 사용자가 많은 현재 시점에 모바일 우선시 하여 코딩 해야한다.
2. 아직 모바일 우선 고려하여 코딩한 경험이 없다.

####논의 결과
* 현재 모바일 퍼스트 시대이기 때문에 모바일을 우선시 하여야 한다.
